## 💡 Sobre o Projeto

O **TechNews** é um portal de notícias de tecnologia desenvolvido inteiramente com HTML5 e CSS3, sem uso de frameworks ou bibliotecas externas. O projeto apresenta um layout moderno com múltiplas seções editoriais, menu de navegação por categorias, barra de busca e destaques organizados por tema.

O site está publicado via **GitHub Pages** e pode ser acessado diretamente pelo navegador.

---

## ✨ Funcionalidades

- [x] Menu de navegação com ícones SVG personalizados
- [x] Barra de busca no header
- [x] Navegação por categorias de tecnologia
- [x] Seção de notícias em destaque (cards com imagem, categoria e título)
- [x] Seção **"Mais lidas da semana"**
- [x] Seção **"Destaques da Inteligência Artificial"** com artigos completos
- [x] Seção **"Viu isso aqui?"** com notícias variadas
- [x] Layout responsivo para desktop e mobile
- [x] Assets visuais próprios (logo, ícones SVG, imagens editoriais)

## 🚀 Como Executar

O projeto **não requer instalação** de dependências. Siga os passos:

### Pré-requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- [Git](https://git-scm.com) para clonar o repositório *(opcional)*

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/toblergabriel/Projeto-Noticias.git

# Acesse a pasta do projeto
cd Projeto-Noticias

# Abra o index.html no navegador
# (duplo clique no arquivo, ou use a extensão Live Server no VS Code)
```

> 💡 **Dica:** A extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) do VS Code permite visualizar o projeto com recarregamento automático ao salvar.

Ou acesse diretamente: **[https://toblergabriel.github.io/Projeto-Noticias/](https://toblergabriel.github.io/Projeto-Noticias/)**
